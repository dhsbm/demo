`animation` + `@keyframes`

```scss
* animation-name: none  // 动画名
* animation-duration: 0s // 持续事件
* animation-timing-function: ease // 运行曲线
* animation-delay: 0s // 延迟时间
* animation-iteration-count: 1 // 执行次数
* animation-direction: normal // 方向
* animation-fill-mode: none // 执行完毕后停留位置
* animation-play-state: running // js 检测动画是否在运行
```
